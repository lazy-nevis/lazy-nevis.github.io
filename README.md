# LazyNevis Landing Page

Marketing site for [LazyNevis](https://github.com/SimStm/lazy-nevis) — privacy-first desktop focus tool.

**Live:** [lazy-nevis.github.io](https://lazy-nevis.github.io)

## Stack

React 19 · TypeScript · Vite · Tailwind CSS v4 · shadcn/ui · Swiper · PhotoSwipe · Canvas mesh hero

## Development

```bash
pnpm install
pnpm dev
```

## Build & deploy

```bash
pnpm build
```

Push to `main` deploys via GitHub Actions.
